## 기능
- 신용카드 정보
    - 카드번호
    - CVC
    - 유효기간
    - 비밀번호
    - 사진 저장

- 은행계좌 정보
    - 보안카드
    - 비밀번호

- 웹사이트 정보
    - ID / 비밀번호

- 개인정보
    - 통관고유부호번호
    - 운전면허증
    - 주민등록증
    - 여권

- 위젯
    - 은행 / 신용카드 정보 바로가기
    - 잠김화면에서 신용카드 바로가기

- URL Scheme
    - coolmintwallet://
