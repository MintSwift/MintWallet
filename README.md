## 배포
 - 2.2.0 ( 2023.01월 말 예정 )
    ```
    [개선]
    - 위젯 사용자 편집 ( # 이영윤 )
    - 개인정보 탭에 커스텀 가능한 항목 추가 ( # 모재혁 )

    [버그]
    - 문구 오류 ( # 조종민 외 1명 ) 
    ```

    ```
    [개선]
    - 신용카드/은행/웹페이지/개인정보 개인별 컬러 ( # 유롤 )

    [버그]
    - 키보드 올라올때, 스크롤 올리기 ( # tachibana ) 
    - 저장 아이콘 '저장'으로 변경 ( # 98percent )
    ```

## 기능
- 신용카드 정보
    - 카드번호
    - CVC
    - 유효기간
    - 비밀번호
    - 사진 저장

- 은행계좌 정보
    - 보안카드
    - 비밀번호

- 웹사이트 정보
    - ID / 비밀번호

- 개인정보
    - 통관고유부호번호
    - 운전면허증
    - 주민등록증
    - 여권

- 위젯
    - 은행 / 신용카드 정보 바로가기
    - 잠김화면에서 신용카드 바로가기

- URL Scheme
    - coolmintwallet://
